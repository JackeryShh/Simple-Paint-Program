This is a simple paint program. It is developed in Visual Studio 2013
with using C++, OpenGL, and Pui library.

The program includes following properties:
• Drawing brush, line, rectangle, circle, 2D polygon
  with mouse.
• Changeable brush size.
• Color chooser with sliders. (R,G,B,A)
• Clearing the screen.
• Saving the work as png file.
• Opening a bmp image file.

For more information about Pui library:
http://plib.sourceforge.net/pui/
